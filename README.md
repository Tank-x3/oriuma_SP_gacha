# オリウマサポートカードガチャ (Beta)

「オリウマ（オリジナルウマ娘）」たちのサポートカードを獲得して収集する、コミュニティ用ファンツールです。
現在は**ベータ版（v0.02）**として公開しています。

## 🐎 このツールについて
このツールは、仮想の「ガチャ」を回して、オリウマたちのカードを集めるシミュレーターです。
ジュエル（課金アイテム）の概念はなく、**誰でも・何度でも・無料で**ガチャを回すことができます。

お気に入りのオリウマが出るまで回したり、珍しい組み合わせの結果をスクショしてコミュニティで共有したりして楽しんでください！

## 🎮 遊び方と見どころ

1. **ガチャを回す**
   - ホーム画面の「10連ガチャを引く！」ボタンを押してください。
2. **演出を楽しむ（New!）**
   - 「アルバム（ファイル）」が表示されます。表紙のセリフや色に注目してください（虹色なら…！？）。
   - タップして開くと、カードの山札が表示されます。
   - **昇格演出:** 銀色（R）のカードだと思っても、タップした瞬間に金（SR）や虹（SSR）に変わることがあります！
   - **カットイン:** 運が良ければ、キャラクターごとの特殊なカットイン演出が入ることもあります。
3. **結果を確認する**
   - 1枚ずつじっくりめくることも、スキップして一覧を見ることも可能です。
   - カードをタップすると、フレーバーテキストなどの詳細が見られます。
4. **アルバムを埋める**
   - メニューの「カードアルバム」から、獲得状況を確認できます。
   - レアリティごとのコンプリート率や、フィルター機能を使って収集状況をチェックしましょう。

## ⚠️ データについて
- **自動保存:** 獲得したカードのデータは、お使いのブラウザに自動的に保存されます。
- **機種変更・バックアップ:**
  - 「データ管理」メニューから、セーブデータを「テキストコード」として発行できます。
  - スマホの機種変更をする際や、念のためにバックアップを取りたい場合は、このコードをコピーしてメモ帳などに保存してください。
  - 別の端末でこのコードを「インポート（読み込み）」することで、続きから遊ぶことができます。

## 🚧 現在のステータス

  - カード本体のイラスト表示は現在準備中です（テキストとアイコンでの表示となります）。今後のアップデートをお待ちください。
- **動作環境:** PC / スマートフォン (iOS, Android) のブラウザで動作します。
